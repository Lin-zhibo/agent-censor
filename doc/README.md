# Agent Censor 工程文档索引

本文档包面向甲方交付，基于 `NOTE.md` 的内容安全审核目标和当前项目目录骨架编写。当前仓库仍处于早期骨架阶段，因此下列内容统一表述为目标设计、建议实现和验收口径，不表示代码已经完成。

## 项目定位

Agent Censor 是面向多业务、多模态内容安全审核的目标系统。系统需要覆盖政治敏感、色情低俗、不良价值观、未成年安全、广告、画风不正、低价值等内容风险，并通过规则引擎、多模态模型、智能体工具调用和 Graph RAG 知识检索形成可解释的审核结果。
